# CodeAlpha Python Projects

This repository contains three Python projects completed as part of the CodeAlpha Internship Program.

## Projects

### 1. Hangman Game
A word guessing game where the player attempts to guess a hidden word one letter at a time before running out of attempts.

#### Features
- Random word selection
- Letter-by-letter guessing
- Input validation
- Win and lose conditions

---

### 2. Stock Portfolio Tracker
A simple Python application that calculates the total value of a stock portfolio based on user input.

#### Features
- Accepts stock symbols
- Accepts quantity input
- Calculates portfolio value
- Displays total investment amount

---

### 3. Basic Chatbot
A simple rule-based chatbot that responds to common user inputs.

#### Features
- Greeting responses
- Basic conversation handling
- User-friendly interaction
- Exit command support

---

## Technologies Used

- Python 3

- Python programming fundamentals
- Loops and conditional statements
- Lists and dictionaries
- User input handling
- Problem-solving and logical thinking
